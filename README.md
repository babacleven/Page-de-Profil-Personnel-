# Page de profil personnel

Cette page présente un profil personnel de façon simple et élégante.

## Description

Ce projet est une page web statique qui affiche :
- une présentation personnelle
- les compétences principales
- une photo ou image de profil
- une mise en page claire et moderne

## Structure du projet

- [index.html](index.html) : le contenu principal de la page
- [images](images) : les images utilisées dans la page

## Comment visualiser le projet

Ouvrez simplement le fichier [index.html](index.html) dans votre navigateur web.

## Comment modifier le projet

1. Ouvrez [index.html](index.html)
2. Modifiez le texte, les sections ou le design
3. Remplacez les fichiers dans [images](images) si nécessaire

## Prérequis

Aucun prérequis n’est nécessaire, car il s’agit d’un projet HTML simple.
